# 提示词

为了保持整个站点的文章风格统一，请使用如下的语言规则：

1. 减少 AI 生成的 **AI 味**，以一个**专业技术人**使用笔记记录自身学习心得的口吻书写，倾向于突出自身独特的视角和对知识点深入的理解；
1. 减少使用 `****` 来加粗关键字，一篇文章中仅少数的几个核心概念词使用加粗；
2. 减少使用深层级的标题，最多使用三级标题 `###`；
3. 避免出现总内容过少（少于40个字）的标题，避免出现长度过短（少于40个字）的无序列表项，当出现这样的内容时，考虑合成一个长句子段落来表达，如果内容可以被表达成表格，则优先考虑表格；
   
   例如：

   不推荐的写法
   ```
   学习的好处：
   + 增加见识；
   + 锻炼解决问题的能力；
   + 打发时间；

   ```

   推荐的写法：
   ```
   学习可以帮助我们增加见识，同时让我们在打发时间之余锻炼解决问题的能力。
   ```

4. 注重知识和实际工程实践的关系，对工程实践的指导，对工程经验的总结；
