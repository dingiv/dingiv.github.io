# 无监督学习

## 无监督学习
无标签，发现数据内在结构。

### 聚类：K-Means
将数据分K群，使群内相似、群间差异大。
- **步骤**：
  1. 随机初始化K个中心。
  2. 分配：每个点到最近中心。
  3. 更新：中心=簇均值。
  4. 收敛或迭代固定次数。
- **距离**：欧氏。
- **优点**：简单、快、可扩展。
- **缺点**：需预设K（肘部法/轮廓系数选）、对初始化敏感（K-Means++改善）、非球形簇差。
- **评估**：轮廓系数、Calinski-Harabasz。

### 降维（Dimensionality Reduction）
压缩高维特征，保留主要信息，缓解维度灾难。
- **PCA（主成分分析）**：
  - 目标：最大方差投影。
  - 步骤：中心化 → 协方差矩阵 → 特征向量 → 选前K维。
  - 线性、无监督。
- **t-SNE**：非线性，适合可视化（低维嵌入）。
- **优点**：降噪、加速训练、可视化。
- **缺点**：PCA信息丢失、t-SNE不适合新数据。
