AI 局限

目前的 AI 大模型是基于概率统计和深度神经网络，不可解释性 -> 研究符号主义 AI
2025 年，可解释 AI (XAI) 市场预计达到 97.7 亿美元，焦点包括注意力机制可视化和因果推理工具，以提升模型透明度。全球政策如欧盟 AI 法案要求高风险模型提供解释路径。
运行功耗太大，算法效率低，本地部署门槛高，依赖远程，远程的话数据隐私安全不行，并且受限于网络环境 -> 这个得慢慢来
AI 数据中心预计 2025 年消耗美国电力 9%。优化如模型量化可降低 LLM 能耗 90%；DARPA 的能量感知机器学习项目针对生成 AI 提升效率，同时保持性能。硬件进步如 NVIDIA 的 AI 优化天气模型减少能源系统漏洞。
无记忆，目前是基于所谓的 model context 这个东西来充当记忆 -> 研究基于状态机和动态神经网络来构建 AI，参照人脑神经元的突触
上下文窗口扩展至 100 万 token（如 Gemini 3），但仍限于短期记忆。新兴研究探索突触启发动态网络，支持连续学习而非静态权重。
没有 学习 -> 试错/验证 -> 调整 -> 再验证 的动态成长能力，这个是人类学习和认识世界的一个非常基本的方式
当前模型缺乏连续学习，导致灾难性遗忘。2025 年进展包括元学习框架，如 MAML，用于在线适应。
没有感官能力，没有手脚，目前是使用 MCP 技术，在 context 中让 AI 分析可以使用的工具 -> 开发 AI 键鼠驱动和高性能截屏驱动，让 AI 能够以人的视角进行 UI 观察和 UI 操作
具身 AI 研究整合多模态 LLM 与机器人控制。工具如 Amazon Bedrock Agents 使用视觉 grounding 实现 UI 交互。

LLM 训练

预训练
自监督学习大规模语料，目标如下一 token 预测。2025 年趋势：稀疏 MoE 架构减少计算 30%。
监督微调
使用标注数据对齐任务，如指令跟随。变体包括 DPO（直接偏好优化），取代部分 RLHF 以简化过程。
强化学习
RLHF（人类反馈强化学习）为主流，结合 PPO 算法。2025 更新：合成数据增强反馈循环，降低人类标注成本。
