# 通信技术

现代计算机系统中的通信技术构成了一个庞杂的家族：从芯片内部的芯粒互联到跨洲际的数据中心网络，从传统的以太网到专用的 GPU 直连总线，不同的通信需求催生了差异巨大的技术方案。

理解这个家族的关键在于两个维度：物理拓扑（设备之间怎么连）和协议分层（数据在各层怎么传）。前者帮助你在装机选型时不犯错，后者帮助你在系统设计时定位瓶颈。

## 内容结构

通信技术的讲解按硬件选型和协议原理两条线展开。

硬件选型线覆盖从机内到机间的各种互联技术：PCIe（通用总线）、NVLink（GPU 直连）、InfiniBand（高性能网络）、以太网（通用网络）、RDMA（远程内存访问）、CXL（缓存一致性互联）等。每种技术从带宽、延迟、成本、适用场景四个维度进行对比。

协议原理线采用分层视角理解通信栈：物理层的信号编码和 SerDes、数据链路层的流控和重传、传输语义层的消息模型与内存模型、缓存一致性层、拓扑层的交换结构、集合通信层的 NCCL/MPI 原语。

这两条线最终汇聚在分布式训练场景中：Megatron 的 3D 并行策略本质上就是把不同通信频率的操作映射到不同物理层和拓扑层。
