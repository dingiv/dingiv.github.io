# 主板

主板（Motherboard）是计算机的核心载体，连接 CPU、内存、显卡、存储和各类外设。主板的设计取向决定了整机的能力上限——消费级主板追求单核性能和性价比，工作站主板追求 PCIe 扩展和内存带宽，服务器主板追求可靠性（RAS）和多路扩展。

## 三级平台

按应用场景，主板平台分为三个层级，对应不同的 CPU 产品线：

| 层级 | 定位 | CPU 产品线 | 典型平台 | 内存通道 | PCIe 通道 | 多路 | ECC | BMC |
| ---- | ---- | ---------- | -------- | -------- | --------- | ---- | --- | --- |
| 消费级 | 个人 PC、游戏 | Ryzen / Core | AM5 / LGA1851 | 2 通道 | 24~28 条 | 否 | 否 | 否 |
| 工作站 | 影视渲染、AI、EDA | Threadripper | sTR5 (TRX50/WRX90) | 4~8 通道 | 48~128 条 | 否 | 部分 | 部分 |
| 服务器 | 数据中心、云计算 | EPYC / Xeon | SP5 / SP3 | 8~12 通道 | 96~160 条 | 是 | 是 | 是 |

三级平台的边界不是绝对的。消费级主板的顶级型号（如 X870E）在内存和 PCIe 扩展上已接近入门工作站；而工作站的 WRX90 在 BMC 带外管理和 ECC 内存支持上已具备服务器级特性。真正的分水岭在于架构取向——消费级以芯片组（PCH）为枢纽管理外设，工作站和服务器以 SoC 形式将 PCIe 根复合体集成进 CPU。

## 主板与内存的约束

主板和内存之间存在严格的适配关系，不同层级平台的约束差异巨大。

| 约束维度 | 消费级 | 工作站 | 服务器 |
|---------|--------|--------|--------|
| 内存类型 | UDIMM | UDIMM/RDIMM | RDIMM/LRDIMM |
| 通道数 | 2 | 4-8 | 8-12 |
| ECC | 部分支持 | Pro 型号支持 | 强制 |
| 最大容量 | ~192 GB | 1-2 TB | 6+ TB |
| XMP/EXPO | 常用 | 部分支持 | 禁止 |
| 填充规则 | 对称即可 | 对称 + RDIMM | 严格对称 + NUMA 感知 |

消费级主要关注 1 DPC vs 2 DPC 对高频稳定性的影响，以及 XMP/EXPO 不等于即插即用。工作站的核心约束是 TRX50 上的 Pro CPU 降配运行——内存通道从 8 条被主板物理走线限制为 4 条。服务器层面涉及通道对称填充、NUMA 交错策略、ECC 故障处理、Rank 数量对频率的影响，以及内存功耗和散热——单台 Genoa 服务器仅内存的持续功耗就可超过 200W。
