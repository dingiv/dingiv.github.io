# 物理分层
物理层在内部主要分成三个层次：

## 协议层
物理层中最靠上的一层，负责把数据变成“比特流”，并进行编码。

功能： 8b/10b 编码、128b/130b 编码、加扰（Scrambling）、成帧。

实例： PCIe 协议就在这里定义了它是 Gen3 还是 Gen4。无论是 M.2 接口还是 PCIe x16 金手指，它们在逻辑物理层上跑的可能都是完全一样的 PCIe 数据流。

1. 电气子层 (Electrical Sublayer) —— “信号怎么跳”
这一层关注的是电压、电流和频率。

功能： 差分信号（Differential Signaling）、电压摆幅、时钟恢复（CDR）。

关键点： PCIe 和 M.2 在这一层几乎是通用的，因为它们都采用 100MHz 的参考时钟和类似的差分对设计。但 RJ45 (以太网) 在这一层完全不同，它需要经过复杂的 PAM4 或 PAM16 调制，电压标准和频率与 PCIe 完全不兼容。

3. 介质依赖子层 (PMD / Physical Connector) —— “长什么样”
这就是你看到的各种接口形状，即物理实体。

功能： 具体的引脚定义、接触件形状、阻抗匹配。

关系：

PCIe 金手指： 专为大功率、多通道（x16）设计的机箱内插拔接口。

M.2 (NGFF)： 专为移动端、紧凑空间设计的接口，虽然走的是 PCIe 的路，但个头小、引脚密。

RJ45： 专为长距离（100米）铜缆设计的接口，带变压器隔离。
