# 主板
主板是承载和连接电脑各个部件的载体，主板还配有大量的微型芯片单元，负责硬件的初始化、供电、协调管理、扩展等重要职责。不论是在嵌入式微型计算机中，还是在高性能服务器中，都需要主板或者其他专用 PCB 来承载连接各个硬件单元。

> PCB（Printed circuit board）：印刷电路板，是电子器件的支撑体，在这其中有金属导体作为连接电子元器件的线路。

## CPU 通道
对于 PC 和服务器的 CPU，主要通道如下：
```
CPU
├── 内存控制器（DDR5）
├── 直连 PCIe 通道（GPU + M.2）
└── 扩展总线通道 → 南桥（芯片组）
```

+ 内存控制器：是 CPU 与内存的高速通道，使用 DDR 总线进行连接；
+ 直连 PCIe 通道：芯片组的速度对于 GPU 和系统盘来说还是太慢了，因此，目前的 CPU 给这两个设备开了后门。前者满足了 GPU 和 CPU 的高效协作，后者极大地优化了电脑开机的速度，从分钟级优化到秒级；
+ 扩展总线通道：连接芯片组，是 CPU 与其他外设进行交互的门户；

## 芯片组
芯片组（Chipset），曾经也叫做南桥，是主板的大脑，由多个芯片单元构成的管理不同的外设通信的枢纽。芯片组通过扩展总线和 CPU 相连，负责总管 CPU 和外设之间的通信。

门前的消费级 PC 芯片组主要支持 PCIe 总线、USB 总线、SATA 总线等等，在嵌入式设备上，I2C 总线、USB 总线还有非总线设备较多；


## 直连 PCIe 通道
直连 PCIe 的通道数量是有限的，消费级 PC 的数量一般也就在 20 条。根据 PCIe 的规范可知，一条 PCIe 5 代通道带宽大约是 4GiB/s，一条 PCIe 4 代通道的带宽大约是 2GiB/s。
