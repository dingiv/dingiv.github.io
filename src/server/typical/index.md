# 典型业务
服务端开发的注重对业务场景的理解和抽象能力，成熟的服务端开发者需要熟练应对不同的业务需求，快速给出合理的方案。本文档整理了服务端开发中的典型业务场景，每个场景都包含核心问题、技术方案和工程实践。

| 场景       | 核心问题                         | 关键技术                                      |
| ---------- | -------------------------------- | --------------------------------------------- |
| 登录鉴权   | 身份验证、状态保持、安全防护     | Session/JWT、OAuth2、SSO、多因素认证          |
| 验证码 | 防刷保护、安全验证、用户体验     | 图形验证码、短信验证码、限流、风控            |
| 支付系统   | 支付对接、回调处理、退款对账     | 支付宝/微信支付、异步回调、签名验证、幂等处理 |
| 订单管理   | 订单状态流转、一致性、幂等性     | 状态机、分布式事务、幂等设计、补偿机制        |
| 内容搜索   | 全文检索、相关性排序、实时更新   | 倒排索引、Elasticsearch、中文分词、热搜统计   |
| 消息通知   | 实时通知、到达率、推送效率       | WebSocket、APNs/FCM、消息队列、推送优化       |
| 评论系统   | 内容审核、敏感词过滤、互动管理   | 敏感词库、内容审核、点赞回复、垃圾评论过滤    |
| 图片上传   | 文件存储、格式转换、CDN 加速     | 对象存储、异步处理、图片压缩、懒加载          |
| 文件下载   | 大数据量、内存控制、异步处理     | 异步导出、流式处理、临时存储、下载通知        |
| 秒杀活动   | 超高并发、库存扣减、防刷保护     | Redis 预减库存、消息队列削峰、分布式锁        |
