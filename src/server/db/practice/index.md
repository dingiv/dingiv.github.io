# 数据库工程实践

理论知识是基础，工程实践是将知识转化为价值的关键。本章节从实际工程角度，介绍数据库相关的规范、容量规划和云服务。

## 内容概览

本章节内容包括：

- **数据库规范**：命名规范、字段设计规范、索引规范、SQL 编写规范
- **容量规划**：存储估算、QPS 评估、扩容预案
- **云数据库服务**：RDS、PolarDB、Cloud SQL、Aurora 对比

## 学习路径

建议按照以下顺序学习：
1. 先了解数据库规范的重要性（数据库规范）
2. 掌握容量规划的方法（容量规划）
3. 了解云数据库服务的特点和选型（云数据库服务）
