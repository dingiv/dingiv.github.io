# 主板

## 常见主板型号
PCIe 通道决定显卡安装的数量；M.2 决定了硬盘安装的数量；

主板和 CPU 一般相适应，并且分为消费级、专业级和企业级，分为对应个人 PC、工作室的工作站、企业的服务器。

intel 的 CPU 分为
+ intel Core
+ intel Xeon
