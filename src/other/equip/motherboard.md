# 主板

| 参数               | 核心含义                           | 为什么关注                                         |
| ------------------ | ---------------------------------- | -------------------------------------------------- |
| 芯片组（Chipset）  | 决定 CPU 支持、PCIe 通道、超频能力 | 错芯片组 = CPU 装不上；高端芯片组解锁更多 PCIe/M.2 |
| CPU 插座（Socket） | 物理接口                           | 插座不匹配 = 无法安装 CPU                          |
| 内存支持           | 类型/频率/通道/容量                | 影响多任务、游戏帧数；DDR5 已主流                  |
| PCIe 插槽          | 版本 × 通道数                      | 显卡/SSD/扩展卡带宽瓶颈                            |
| M.2 插槽           | SSD 接口数量/规格                  | 系统盘+游戏盘速度                                  |
| 供电相数（VRM）    | CPU 供电模块                       | 超频/高功耗 CPU 稳定性                             |
| 后置 I/O           | USB/网口/音频                      | 外设连接                                           |
| 扩展槽             | SATA/PCIe 小槽                     | 机械硬盘/声卡/采集卡                               |
| BIOS 功能          | 超频/调试                          | 后期优化                                           |
| 尺寸/布局          | Form Factor                        | 机箱兼容 + 走线                                    |

优先选择 CPU，然后根据 CPU 选择相应的主板，主要分为 intel 和 AMD 的主板两种。

| 场景                 | 错误                  | 后果      | 正确做法            |
| -------------------- | --------------------- | --------- | ------------------- |
| 买 Z790 配 i5        | 超频芯片组浪费        | 多花 500+ | B760 够用           |
| B650 缺 PCIe 5.0 M.2 | 未来 SSD 降速         | 升级受限  | 选 B650E 或 X670    |
| 忽略 VRM             | 廉价 8 相板配 Ryzen 9 | 掉频/死机 | 看 TechPowerUp 评测 |
| WiFi 版多花 300      | 有线够用              | 浪费      | 单独买 PCIe 网卡    |

## 3 步选主板
1. 定 CPU → 查插座 + 推荐芯片组（官网兼容列表）。
2. 列需求：M.2 数量？超频？RGB？预算？
3. 用 PCPartPicker 过滤 → 看用户评价 VRM 散热 + 售后（华硕/微星/技嘉/华擎）。

Intel CPU 和 AMD CPU 必须使用完全不同的主板，不能混用。
核心原因：插座（Socket）物理接口不同

## pcie 通道
PCIe 5.0 通道参数详解（以 “CPU: x16 + x4 / 芯片组: x4” 为例）
PCIe（Peripheral Component Interconnect Express）是高速串行总线，用于连接显卡、SSD、网卡等。
5.0 = 第 5 代，单通道带宽 4GB/s（x1 = 4GB/s，x16 = 64GB/s）。
参数格式：来源 : 版本 x 通道数（lanes）。


来源,参数,含义,实际用途,带宽（PCIe 5.0）
CPU,x16,CPU 直连 16 条 PCIe 5.0 通道,显卡主槽（GPU 全速）,64 GB/s
CPU,+ x4,CPU 额外直连 4 条 PCIe 5.0 通道,"高速 M.2 SSD（系统盘）
或第 2 张显卡（x4 模式）",16 GB/s
芯片组,x4,芯片组 通过 DMI 4.0 x8 连接 CPU，再分出 4 条 PCIe 5.0,"第 2/3 个 M.2
或扩展卡（网卡、声卡）",16 GB/s（共享）


## M.2 插槽数
M.2 数量 = 主板上 物理 M.2 插槽 的总数。
每个插槽可装 1 个 NVMe SSD（或极少数 SATA SSD/WiFi 卡）。
核心意义：决定你能同时装几个高速 SSD（系统盘 + 游戏盘 + 数据盘）。
