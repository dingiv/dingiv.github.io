# AI 模型硬件

根据 Deepseek 蒸馏模型档位来进行划分：

| 参数规模 | 训练显存 | 推荐配置                         |
| -------- | -------- | -------------------------------- |
| 70B      | 182GiB   | 2 路 numa + rtx PRO 4000 24G * 8 |
| 32B      | 84       | 2 路 numa + rtx 3080 20G * 10    |
|          |          | 2 路 numa + rtx 5060Ti 16G * 8   |
| 14B      | 37       | 消费级 rtx4090 48G、工作级 rtx   |
| 8B       | 21       | 消费级 rtx3090 24G、rtx4090 48G  |
| 1.5B     | 4        | 消费级 PC                        |

训练使用的显存 = 参数规模 * 2 GiB/B * 1.3
部署使用的内存 = 参数规模 * 1 GiB/B * 1.3

部署时所需要的显存是训练时的一半。
```
// 70B
70B * 2GiB/B * 1.3 = 182 GiB
```
