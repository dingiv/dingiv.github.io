# 前端工程化
目前的前端工具主要是 webpack，不过目前 Vite 的发展增速非常之快，而 webpack 在2023年的 JavaScript State 调查中的使用意愿很低，使用人数在逐步下降，基本可以说是被 Vite 把市场份额抢走了。

目前的 React 的 create-react-app 脚手架已经停止了维护，React 社区的人纷纷开始转向 Vite；而 Vue 官方是强推 Vite；Webpack 的作者也另寻出路使用 rust 重写了一个性能更加高效的 turbopack，并在当红全栈框架 Next.js 中默认集成。
